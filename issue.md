# Issues

## Dir選択：深い階層へのナビゲーション対応

### 背景

edit画面のDir選択は現在1階層のみ表示。深いパスを指定したい場合に対応できていない。

### 提案：ドリルダウン型ディレクトリブラウザ

- `Enter` or `→` で選択中のディレクトリに入る
- `..` エントリで1つ上に戻る（プロジェクトルートより上は表示しない）
- フォームの次フィールドへの移動は `Tab` など別キーに分ける

```
# 例：プロジェクトルートにいる場合
Dir  [ .    ]
     [ src  ]  ← Enterで入る
     [ docs ]

# src に入った後
Dir  [ ..      ]  ← Enterで戻る
     [ src     ]  ← 現在地（このままTabで確定）
     [ src/api ]
     [ src/ui  ]
```

### 代替案

- リスト選択に加えて `e` キーでテキスト直接入力モードに切り替える（パスを知っている場合に速い）
