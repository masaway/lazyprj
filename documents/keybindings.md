# キーバインド一覧

`?` キーでアプリ内からもヘルプオーバーレイを表示できます。

---

## メイン画面

| キー | 動作 |
|------|------|
| `j` / `↓` | カーソル下移動 |
| `k` / `↑` | カーソル上移動 |
| `g` | 先頭へ移動 |
| `G` | 末尾へ移動 |
| `1` | Projects パネルにフォーカス |
| `2` | Detail パネルにフォーカス |
| `Tab` | Projects / Detail パネルのフォーカス切替 |
| `Enter` / `s` | セッション起動、または起動中ならアタッチ |
| `d` / `x` | セッション停止 |
| `R` | セッション再起動（確認ダイアログあり） |
| `e` | エディタ画面を開く（オーバーレイ表示） |
| `c` | 新規セッション作成（クイックスタート、オーバーレイ表示） |
| `a` | 自動起動フラグのトグル（`★` 表示、即保存） |
| `A` | `★` のついた全プロジェクトを一括起動 |
| `X` | プロジェクトをスキップ済みへ移動 |
| `n` | スキャン画面を開く（オーバーレイ表示） |
| `S` | スキャンディレクトリ設定画面を開く |
| `r` | プロジェクト一覧・セッション状態を再読み込み |
| `?` | ヘルプオーバーレイを表示 |
| `q` / `Ctrl+C` | 終了 |

---

## 新規セッション作成画面（`c` キーで起動）

メイン画面の上にオーバーレイ表示されます。任意のディレクトリを指定してtmuxセッションを作成・登録します。

| キー | 動作 |
|------|------|
| `Tab` / `Shift+Tab` | フィールド切替（ディレクトリ → セッション名） |
| `Enter` | セッションを作成して登録 |
| `Esc` | キャンセルしてメイン画面に戻る |

> セッション名を空欄にしたままTabを押すと、ディレクトリ名から自動補完されます。

---

## エディタ画面（`e` キーで起動）

メイン画面の上にオーバーレイ表示されます。ウィンドウとペインの追加・編集・削除ができます。

| キー | 動作 |
|------|------|
| `j` / `k` | ウィンドウ／ペイン移動 |
| `Tab` | Windows / Panes パネル切替 |
| `a` | ウィンドウ／ペインを追加 |
| `e` | 選択中の項目を編集 |
| `d` | 選択中の項目を削除 |
| `w` | 設定を保存 |
| `Esc` | メイン画面に戻る |

### ペイン編集フォーム

| キー | 動作 |
|------|------|
| `j` / `k` | Dirリスト内を移動 |
| `e` | Dirを直接テキスト入力 |
| `Tab` / `Shift+Tab` | フィールド切替（Dir → Command → Execute） |
| `Space` | Execute フラグのON/OFF切替（Executeフォーカス時） |
| `Enter` / `w` | 保存 |
| `Esc` | キャンセル |

### ウィンドウ編集フォーム

| キー | 動作 |
|------|------|
| `Tab` / `Shift+Tab` | フィールド切替（名前 → レイアウト） |
| `←` / `→` | レイアウト選択（レイアウトフォーカス時） |
| `Enter` / `w` | 保存 |
| `Esc` | キャンセル |

---

## スキャン画面（`n` キーで起動）

| キー | 動作 |
|------|------|
| `j` / `k` | カーソル移動 |
| `Space` | 選択トグル |
| `a` | 全選択 / 全解除 |
| `x` | スキップ済みに追加（スキップ済みアイテム上では解除） |
| `Space` / `Enter`（ヘッダー行） | スキップ済みセクションの展開 / 折りたたみ |
| `Enter` / `s` | 選択プロジェクトを追加して保存 |
| `Esc` / `q` | メイン画面に戻る |

スキップしたプロジェクトは `▶ スキップ済み (N)` セクションに格納され、設定ファイルの `skipped_paths` に記録されます。次回スキャン時に自動的に除外されます。
